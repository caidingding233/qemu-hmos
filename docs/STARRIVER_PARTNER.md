# 致谢与开源声明｜漫天星光，因你璀璨

今天，满天星光已经汇聚成璀璨星河。

没有人能够熄灭满天星光——每一位开发者、每一次提交、每一行代码，都是我们要汇聚的星星之火。  
星星之火，可以燎原；点点微光，终将照亮前路。

`QEMU-HMOS（Aetherium）` 能在鸿蒙设备上把虚拟机体验从“想象”落到“可用”，不是因为我们有多强，而是因为我们站在巨人的肩膀上，身后还有你们：上游项目、维护者与贡献者。  
你们把时间写进代码，把热爱写进社区，也把未来写进了这片星河。

在此，我们以最郑重的方式致谢，并作出清晰、可追溯的开源声明：

- **合规开源**：严格遵循所有上游项目的许可证条款、版权声明与分发要求。
- **来源透明**：对依赖与参考对象进行明确标注，便于追溯、审计与复现。
- **持续回馈**：持续关注上游更新，及时同步修复与改进，力所能及回馈社区。

## 星光致谢清单（按项目）

### 1. QEMU - Quick Emulator

**项目地址**: https://www.qemu.org/  
**许可证**: GPL-2.0 / LGPL-2.1  
**用途**: 虚拟机核心引擎，提供 ARM64 虚拟化能力  
**致谢**: 感谢 QEMU 社区用长期主义打磨出强大的虚拟化底座，让我们得以在 HarmonyOS 设备上探索运行 Windows 和 Linux 虚拟机的可能性。

---

### 2. LibVNC - VNC 服务器和客户端库

**项目地址**: https://github.com/LibVNC/libvncserver  
**许可证**: GPL-2.0  
**用途**:
- VNC 服务器：用于虚拟机安装阶段的远程桌面显示
- VNC 客户端：用于在 HarmonyOS 应用中显示虚拟机画面

**致谢**: 感谢 LibVNC 提供稳定、跨平台的 VNC 能力，让“装得上、看得见、连得稳”的体验成为可能。

---

### 3. FreeRDP - 远程桌面协议客户端

**项目地址**: https://www.freerdp.com/  
**许可证**: Apache License 2.0  
**用途**:
- RDP 客户端：用于虚拟机运行阶段的远程桌面连接
- 提供比 VNC 更好的性能和用户体验

**致谢**: 感谢 FreeRDP 把复杂协议做成可靠工程，让我们能把更流畅、更清晰、更低延迟的远程桌面体验带给用户。

---

### 4. OpenSSL - 加密工具库

**项目地址**: https://www.openssl.org/  
**许可证**: Apache License 2.0 / OpenSSL License  
**用途**:
- 为 FreeRDP 提供 TLS/SSL 加密支持
- 为网络通信提供加密功能

**致谢**: 感谢 OpenSSL 在底层守护连接安全，让每一次通信都更可信、更可依赖。

---

### 5. GLib - C 工具库

**项目地址**: https://wiki.gnome.org/Projects/GLib  
**许可证**: LGPL-2.1  
**用途**:
- QEMU 的核心依赖库
- 提供数据结构、线程、内存管理等基础功能

**致谢**: 感谢 GNOME 的 GLib 作为基础设施中的基础设施，让上层能力更稳、更快、更易维护。

---

### 6. PCRE2 - Perl 兼容正则表达式库

**项目地址**: https://www.pcre.org/  
**许可证**: BSD License  
**用途**:
- GLib 的依赖库
- 提供正则表达式匹配功能

**致谢**: 感谢 PCRE2 以高性能与高质量实现支撑关键依赖链条，让细节之处同样可靠。

---

### 7. Pixman - 像素操作库

**项目地址**: https://cairographics.org/pixman/  
**许可证**: MIT License  
**用途**:
- QEMU 图形渲染的核心库
- 提供像素级别的图像操作和合成功能

**致谢**: 感谢 Pixman 在像素与合成层面的深厚积累，让渲染链路更高效、更可控。

---
